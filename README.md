# React Portfolio

A responsive portfolio built using React.js and Tailwind CSS.

## Live Preview

Check out the live preview [here](https://maricelyvillalobs.netlify.app/).

## Installation

1. Install dependencies:
    ```bash
    npm install
    ```
2. Start the development server:
    ```bash
    npm run dev
    ```
